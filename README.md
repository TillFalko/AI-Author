# AI-Author
A graphical front-end for AI-Based text generation using OpenAI's GPT-2 architecture.

This project is based on [aitextgen](https://github.com/minimaxir/aitextgen) by [Max Woolf](https://github.com/minimaxir).


**Warning**:
This code is **not** maintained and there will be **no** installation and usage support.
